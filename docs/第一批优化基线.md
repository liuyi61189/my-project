## 已知基线问题

1. 文本主要是 UTF-8；旧版 PowerShell 未指定 UTF8 编码时可能显示乱码。
2. 宿主进程的 DEBUG 环境变量可能覆盖 .env 并导致 Django 配置加载失败。本批只报告，不自动修改。
3. JWT、Token、Session 与全局 API CSRF 豁免并存，后续需逐步收紧。
4. 匿名埋点需要限流和防滥用措施。
5. 外部 URL、ADB 和子进程需要专项检查 SSRF、路径及命令边界。
6. backend/apps 是重复旧目录，确认无引用前不删除。
